## Projects in this site

This is only a summation. More elaborate docs are not publicly available anymore since I ditched my personal website. I still update them when needed, but more as an exercise for myself

* LinterGui, a frontend for Python Linters (because I'm too lazy to call them from the command line)
* FileFindr, a tool to Find and Replace in File (because SciTE didn't do the latter)
* LogViewer, a tiny webapp to view server logs in the browser (seemed like a good idea at the time)
* Actiereg, a rewrite of an old issue tracking app that finally has replaced my Trac installation
* Axe, my take on an XML editor
* Ashe, my take on an HTML editor
* CSSEdit, my take on a CSS editor, in the same vein as the previous two
* a-propos, a productivity helper I wrote for work where I can't use it anymore
* notetree, a-propos but in a tree interface and with keywords
* treedocs, rich text notes/docs in a tree interface, with multiple hierarchies possible
* compare-tool, for out-of-sequence comparisons of settings, XML and HTML, plain text and even Python code
* hotkeys, keep track of useful keycombos for all your applications
* Rst2HTML, maintaining web pages in your browser using ReST

All of the above are tools I use on a regular to from-time-to-time basis, some very frequenly and others occasionally. As such, they are updated when I find something that doesn't work or could work better. They're not perfect, but they do the job the way I want them to.
